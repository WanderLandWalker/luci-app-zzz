zzz OpenWrt APK Feed

OpenWrt 25.x APK 软件包仓库，每次代码推送后自动构建。

使用方法

在路由器上执行以下命令添加本仓库：

x86_64

echo "https://Dichgrem.github.io/luci-app-zzz/x86_64" \
    >> /etc/apk/repositories
wget -O /etc/apk/keys/zzz.pub \
    https://Dichgrem.github.io/luci-app-zzz/x86_64/key.pub.pem
apk update && apk add luci-app-zzz

aarch64_cortex-a53 (MediaTek Filogic)

echo "https://Dichgrem.github.io/luci-app-zzz/aarch64_cortex-a53" \
    >> /etc/apk/repositories

mipsel_24kc (Ramips MT7621)

echo "https://Dichgrem.github.io/luci-app-zzz/mipsel_24kc" \
    >> /etc/apk/repositories
